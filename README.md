“Capitales” est un mini‑projet immersif présentant trois grandes villes du monde à travers un slider vertical synchronisé avec un slider d’images. L’objectif : offrir une expérience fluide, élégante et narrative, où le texte et l’image avancent ensemble pour raconter l’essentiel de chaque capitale.

✨ Fonctionnalités
Slider vertical pour le texte (Swiper.js)

Slider d’images synchronisé

Navigation via molette, flèches ou glissement

Effet fade sur les images

Particules animées (Particles.js) pour une ambiance vivante

Design diagonal avec clip‑path pour une composition moderne

Responsive : adaptation complète sur mobile

📸 Aperçu
Un slider immersif où chaque capitale apparaît avec son image, son titre et un court texte descriptif, le tout dans une ambiance élégante et moderne.

💬 Note personnelle
Ce projet m’a permis d’explorer la synchronisation de sliders, les effets visuels avancés (clip‑path, particules, gradients), ainsi que la création d’une ambiance immersive autour d’un thème simple : découvrir trois capitales en allant à l’essentiel.
